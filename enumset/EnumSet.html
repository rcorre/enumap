    Struct EnumSet

    A structure that maps each member of an enum to a single value.

    An EnumSet is a lightweight alternative to an associative array that is useable when your key type is an enum.

    It provides some added benefits over the AA, such as array-wise operations, default values for all keys, and some nice opDispatch based syntactic sugar for element access.

    Constructors

    Name Description
    this Construct an EnumSet from a static array.
    this Construct an EnumSet from an associative array.

    Templates

    Name Description

    Parameters

    NameDescription
